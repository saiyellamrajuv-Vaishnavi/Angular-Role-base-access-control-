* Role level Access Control - Angular UI

Angular 18  application for managing role-based filed access permissions using Angular Material

* Features

1.Search access controls by Role and Access Control Type
2.Add/remove Case Categories and Sub-categoris using chips
3.View results in an Angular material
4.Row-level view and edit actions
5.Reactive Forms with validation

*Tech Stack

1.Angular 18
2.Angular Material 18
3.Typescript
4.Reactive Forms
5.SCSS

*Run the Application

bash
npm install
ng serve

Open:http://localhost:4200

*Build

bash
ng build

# Notes

1.Role and Access Control type are mandatory fields
2.Search displays matching access contol records
3.clear resets the form and hides the results table
